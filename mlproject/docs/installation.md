# Installation

## Stable release

To install mlproject, run this command in your terminal:

```sh
uv add mlproject
```

Or if you prefer to use `pip`:

```sh
pip install mlproject
```

## From source

The source files for mlproject can be downloaded from the [Github repo](https://github.com/Rakeshavs/mlproject).

You can either clone the public repository:

```sh
git clone https://github.com/Rakeshavs/mlproject
```

Or download the [tarball](https://github.com/Rakeshavs/mlproject/tarball/main):

```sh
curl -OJL https://github.com/Rakeshavs/mlproject/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd mlproject
uv sync
```
